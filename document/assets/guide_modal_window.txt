Paso 1: Crear la vista del login en Django
En tu vista de Django, puedes verificar si el usuario y la contraseña son correctos. Si no lo son, pasarás una variable de contexto al template indicando el error.

views.py
python
Copiar código
from django.contrib.auth import authenticate, login
from django.shortcuts import render, redirect

def login_view(request):
    if request.method == 'POST':
        username = request.POST['username']
        password = request.POST['password']
        
        # Verificar las credenciales del usuario
        user = authenticate(request, username=username, password=password)
        
        if user is not None:
            # Si las credenciales son correctas, iniciar sesión y redirigir al menú principal
            login(request, user)
            return redirect('menu_principal')  # Reemplaza 'menu_principal' por el nombre de tu vista
        else:
            # Si las credenciales son incorrectas, renderiza la página con un indicador de error
            return render(request, 'login.html', {'error': True})
    
    return render(request, 'login.html')
En esta vista:

Si los datos ingresados son correctos, se redirige al menú principal.
Si los datos son incorrectos, se recarga la página de login y se pasa un contexto error=True.

Explicación del template:
Formulario de login: Es el formulario que envía el nombre de usuario y la contraseña al servidor.
Ventana modal de Bootstrap: Se define una ventana modal que se activará si las credenciales son incorrectas.
JavaScript: El código JavaScript verifica si la variable error es True. Si es así, muestra la ventana modal automáticamente cuando la página se recarga.

Resumen del flujo:
El usuario ingresa sus credenciales en el formulario de login.
Si las credenciales son incorrectas, se vuelve a cargar la página de login y se pasa un contexto con error=True.
El código JavaScript detecta el valor de error y muestra una ventana modal con un mensaje de error.
Si las credenciales son correctas, el usuario es redirigido al menú principal.
Con esto deberías poder mostrar un modal de error en caso de que el inicio de sesión falle. Si tienes más dudas o algo no funciona como esperas, avísame y lo revisamos.
